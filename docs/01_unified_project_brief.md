## Unified Research Focus
This project studies how AI can support learning by inferring user functional needs and preferences from observable signals (instead of diagnostic labeling), with attention to both neurotypical and neurodivergent users.

## Two Core Questions (must be answered before Experiment 1A full launch)
1. Is there existing evidence that AI can identify neurodivergent users or their functional needs in learning/HCI contexts?
2. Is there existing evidence that AI can identify user preferences (for neurotypical and neurodivergent users) to adapt content or interface?

## Scope Boundary (important)
- In scope:
  - Functional-needs inference
  - Preference modeling
  - Adaptive support in learning/HCI
- Out of scope:
  - Clinical diagnosis claims
  - Medical-only diagnostic pipelines without learning adaptation

## Experiment Structure (current)
- Pretest:
  - Personality: Big Five, MBTI, Type A/B, proactiveness
  - Neurodiversity inclination and media preference (color, figure/text/picture/sound, intensity, speed, attention length)
- Experiment 1A (task-based):
  - AI-assisted learning task
  - Response log, test performance, camera-based eye-tracking
- Experiment 1B (interactive):
  - Group dynamics: proactiveness, control, aggressiveness
  - Outcomes: achievement and conflict
- Post-test:
  - Self-efficacy
  - Learning effectiveness
  - Learning wellbeing
