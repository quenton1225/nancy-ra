# Pre-Meeting Deliverables Checklist (DU Mukun)

## Mandatory Outputs
- [ ] Unified project brief (1 page)
- [ ] Scholar search log and extraction table (first round)
- [ ] Preliminary mini-review (2 pages)
- [ ] Recommendation slide (Route A vs Route B)

## Suggested 6-Day Sprint

### Day 1
- [ ] Finalize term harmonization using two Notes
- [ ] Confirm RQ1/RQ2 wording
- [ ] Start Set A/B/C query runs

### Day 2
- [ ] Complete first 50-result scan for each query set
- [ ] Fill extraction table with first 30 candidates
- [ ] Remove obvious out-of-scope studies

### Day 3
- [ ] Deep-read top 15-20 papers
- [ ] Extract methods, signals, targets, metrics
- [ ] Draft evidence map: direct vs adjacent vs background

### Day 4
- [ ] Reduce to 12-18 core papers
- [ ] Write key findings and gaps
- [ ] Draft recommendation for experiment implications

### Day 5
- [ ] Produce 2-page mini-review
- [ ] Produce 1-page action proposal for next phase

### Day 6
- [ ] Build 10-minute meeting narrative
- [ ] Rehearse and stress-test weak claims

## 10-Min Meeting Script
1. What we asked (RQ1/RQ2)
2. What evidence we found
3. What is still missing
4. What we should do next (Route A or Route B)

## Stoplight Risk Check
- Red: no robust direct evidence
- Yellow: evidence exists but mostly adjacent
- Green: multiple directly relevant studies with reusable methods
